## Docker compose workflow (WIP)

- Create secrets file
```bash
echo "SECRET_KEY_BASE=<some secret>" > secrets.env
```

- Init the database
```bash
docker-compose build && docker-compose run data_dashboard_web rake db:create db:migrate
```

- Bring up the database and the web server
```bash
docker-compose up

```
